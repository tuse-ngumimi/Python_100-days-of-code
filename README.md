# 🐍100 days of Python Mini Projects

This repository contains a collection of small Python projects I’ve worked on to practice core programming concepts, including loops, conditionals, functions, file handling, and more.  
Each folder represents a separate project with its own code and README (where applicable).

---

## 📂 Projects Included

### 1. 🧮 Calculator
- A simple command-line calculator.
- Supports addition, subtraction, multiplication, and division.
- Shows basic function usage.

### 2. 🕹️ Rock Paper Scissors Game
- A simple, yet interactive command-line version of the popular game.
- Makes use of the random module to enable the computer to make a random choice.
- Makes use of `nested if statements`, the `f-string`, and `lists`.

### 3. 📁 File Explorer (Mini CLI Tool)
- Reads file metadata such as created/modified times.
- Handles errors (e.g., `PermissionError`, `NotADirectoryError`).
- Demonstrates working with `os`, `pathlib`, and `datetime`.
